# 📖 Overview

ExpenseFlow Pro is a modern personal finance management application designed to help users organize their finances with ease.

The application provides secure authentication, cloud synchronization, beautiful dashboards, interactive analytics, savings goal tracking, and AI-ready financial insights—all through an elegant and responsive interface.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/rishizz709/expense-tracker.git
```

## Navigate

```bash
cd expense-tracker
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Configure Environment

Create

```text
.streamlit/secrets.toml
```

Add

```toml
SUPABASE_URL="YOUR_SUPABASE_URL"

SUPABASE_KEY="YOUR_SUPABASE_ANON_KEY"
```

## Run Application

```bash
streamlit run app.py
```
